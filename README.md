# ZB - Floating Dictionary

A minimalist, interactive web application where words become physical objects. Type a word, watch it drift, and see it join your collection.

## 🚀 Features

- **Dynamic Word Launching:** Words drift and bounce around the screen using a custom JavaScript physics loop.
- **Color Themes:** Choose from 6 different color presets to categorize or stylize your entries.
- **Auto-Archiving:** Words stay active for 10 seconds before "popping" and moving into the permanent dictionary storage.
- **Responsive Design:** Built with the 'Inter' typeface and a mobile-first UI that works perfectly on desktop or touch devices.

## 🛠️ How to Use

1. **Select a Color:** Use the circular color toggles at the bottom to choose your theme.
2. **Type & Launch:** Enter a word in the input field and hit the "Send" icon or press `Enter`.
3. **View Dictionary:** Click the **Stored** button in the top right to view all the words you have collected during your session.

## 🌐 Deployment

This project is designed to be hosted as a single-page application. 

**To host on GitHub Pages:**
1. Rename the main file to `index.html`.
2. Push the code to a GitHub repository.
3. Enable **GitHub Pages** in the repository settings.
